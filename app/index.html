Current Project:

Hybrid HTML5 mobile app for Android/iOS that I developed for the medical marijuana industry. The app is a single page web application I created with AngularJS and the Apache Cordova/PhoneGap framework. The template consumes back-end data via AJAX request to give the user a current possession limit, itemized menu, and ID-card expiration countdown. Additional features include: google maps API, device offline event listener + error message, expired card lockout, card number login validation, login error messages, and push notifications.

Prior to the start of this project, I had been coding for less than 5 months and was less than 2 weeks into an online AngularJS course. I was able to teach myself enough Angular to create a functional demo of the app in less than a week, and secure a first client.
